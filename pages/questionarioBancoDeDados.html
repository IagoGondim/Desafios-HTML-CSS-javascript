Questionário banco de dados

O que é uma tabela?
É uma estrutura de dados do banco da dados relacional que organiza os dados em linhas e colunas, tendo um nome exclusivo e usada para armazenar informações sobre um tipo especifico de entidade.
O que é um registro?
Conhecido como uma linha, é uma entrada individual em uma tabela. Ele contém informações específicas sobre um campo da entidade que a tabela representa.
O que é chave primária?
É um atributo em uma tabela com um valor único e não pode ser nula e nem pode se repetir, por exemplo a coluna id, que toda tabela tem que ter para identificação.
O que é chave estrangeira?
Uma chave estrangeira é um atributo em uma tabela que estabelece um relacionamento com a chave primária de outra tabela. Permitindo que dados dados de tabelas diferentes se relacionem.
O que é relacionamento?
Um relacionamento em um banco de dados refere-se à associação entre entidades ou tabelas. Usando chaves estrangeiras para conectar registros em uma tabela a registros de outra tabela
O que é cardinalidade? Quais os tipos que existem?
Cardialidade refere-se ao número de instâncias em uma entidade que podem estar associadas a um número especifico. Os tipos são:
  • Um para Um (1:1)
  • Um para Muitos (1:N)
  • Muitos para Um (N:1)
  • Muitos para Muitos (N:N)
O que é entidade associativa e para que serve?
Uma entidade associativa é uma tabela adicional usada para resolver relacionamentos de muitos para muitos (N:N) entre outras tabelas. Atuando como intermediário para conectar as entidades relacionadas, armazenando pares de chaves estrangeiras das entidades relacionadas.
O que é normalização e qual sua finalidade?
É o processo de organizar os dados em um banco de dados de maneira eficiente, elimenando redundâncias e evitando problemas de integridade. Reduzindo a redundância de dados, melhorar a integridade dos dados e simplificar as operalções de consulta e atualização.
